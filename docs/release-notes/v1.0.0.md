# 即梦AI视频CLI工作流 v1.0.0

首个开源版本，把即梦官方 `dreamina` CLI 包装成桌面端 AI 视频工作流。

## Highlights

- LLM 自动生成镜头级分镜脚本
- 调用即梦 CLI 批量生成分镜图片
- 图片审核模块：查看大图、编辑提示词、二次生成
- 确认图片后再进入图生视频，降低积分浪费
- 本地多账号 CLI Router：账号切换、积分刷新、自动选择可用账号
- 项目库与导演模式：管理历史项目，单镜头图片/视频重生成并备份旧素材
- 轻量剪辑器：素材导入、时间线排列、简单转场和 MP4 导出
- macOS / Windows 桌面安装包

## Requirements

- 安装并登录即梦官方 `dreamina` CLI
- 配置 OpenAI 兼容 LLM API
- 视频剪辑导出需要本机可用的 FFmpeg / ffprobe

## Notes

- 本项目为非官方工具，不隶属于即梦、剪映或字节跳动。
- 安装包作为 Release 附件分发，源码仓库不包含构建产物、账号登录态或 API Key。
- macOS 包未签名时可能需要在系统安全设置中允许打开。
- Windows 安装包可能在早期触发 SmartScreen，这是新发布应用常见现象。
